    StayFind

    A full stack hotel booking website using the MERN stack (MongoDB, Express.js, React, Node.js).

    • Technologies

    Description

    StayFind is a comprehensive hotel booking platform that allows users to search, view, and book hotels. It includes user authentication, hotel listings, booking management, and more.

    Features

    • User authentication (sign up, login, logout)
    • Search and filter hotels
    • View hotel details
    • Book a hotel
    • User profile management
    • Admin panel for managing hotels and bookings

    Technologies

    • Frontend: React, Redux, Bootstrap
    • Backend: Node.js, Express.js
    • Database: MongoDB
    • Authentication: JWT (JSON Web Tokens)
    • Deployment: Heroku, Netlify

    Usage

    Once the server and client are running, you can access the application at http://localhost:3000.

    • Homepage: View a list of hotels.
    • Search: Use the search bar to find hotels.
    • Hotel Details: Click on a hotel to view its details.
    • Booking: Book a hotel by selecting the dates and confirming the booking.
    • User Profile: Manage your profile and view your bookings.

    API Reference

    Authentication

    • POST /api/auth/register: Register a new user.
    • POST /api/auth/login: Login a user.

    Hotels

    • GET /api/hotels: Get a list of all hotels.
    • GET /api/hotels/:id: Get details of a specific hotel.
    • POST /api/hotels: Add a new hotel (Admin only).
    • PUT /api/hotels/:id: Update hotel details (Admin only).
    • DELETE /api/hotels/:id: Delete a hotel (Admin only).

    Bookings

    • GET /api/bookings: Get a list of all bookings.
    • GET /api/bookings/:id: Get details of a specific booking.
    • POST /api/bookings: Create a new booking.
    • DELETE /api/bookings/:id: Cancel a booking.
